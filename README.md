SATELLISENSE-AI-Satellite-Image-Analysis
An AI-driven dashboard for satellite image analysis using Large Language Models to generate insights, visualizations, and automated reports.




## Repository Contents
- `app.py` – Core application logic
- `README.md` – Project overview
- `PROJECT_EXPLANATION.md` – Detailed system explanation
- `FEATURE_SUMMARY.md` – Features, novelty, and real-world impact
- `NEW_FEATURES_ADDED.md` – Enhancements and advanced AI capabilities
- `TEACHER_PITCH.md` – Simplified explanation for evaluations
- `Satelli.PPT.pptx` – Project presentation



## Data and Security Note
The dataset used during development is not included in this repository due to data size and ownership considerations.  
API keys are excluded and must be configured locally.
